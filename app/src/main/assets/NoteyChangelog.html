2.2.1 - [2015-05-09]

• Bug fix: Having Expandable Notifications unchecked caused title to take over note text after reboot.
• Bug fix: Having Expandable Notifications unchecked caused icons shown as white in notification tray.
• Bug fix: Mic input now goes to note text box as default.
• Removed option for Pro users to choose their icons.
• Pro users can install Notey on multiple devices and all devices will be Pro.

 

2.2 - [2015-03-04]

• Introducing Notey Pro!  Upgrade to Notey Pro for alarm sounds, 5 extra icon choices, and notification quick shortcut.
• Fix for only being able to create one alarm - now all your notes can each have an alarm.
• Crash fixes.
• More Material goodness.
• Settings and About menus visual enhancements.
 - Removed hidden row option and long press for voice input option.
 - Added link to GitHub.
• New logo to celebrate Notey Pro.
• Minor fixes:
 - Changed "00 AM" to "12 AM"
 - When creating new alarm, time displayed is now auto set to one minute ahead of current time.
 - Alarm date/time text format is shorter.
 - Info Screen now shows icon and alarm if one is set. Click on the alarm time to go directly to the alarm edit window.
 - Ticker text added to notifications.
 - Mic input will now go to which text box was selected.
